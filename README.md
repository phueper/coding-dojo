## Installation

```bash
$ npm install
$ cp -r lib/template lib/<target>
```

## Verwendung
```bash
$ ./test.sh lib/<target>
```

## IntelliJ Settings

1. neues Projekt
2. static Web auswählen
3. Projektverzeichnis wie oben auswählen
4. Terminal von IntelliJ starten
